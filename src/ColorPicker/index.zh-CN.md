# ColorPicker 颜色选择器

ColorPicker 是一个支持多种模式的颜色选择组件，基于 react-color 实现，提供了丰富的颜色选择器样式和交互方式，可满足不同场景下的颜色选择需求。

## 组件特性

- 🎨 支持多种模式的颜色选择面板
- 🧰 内置 Sketch 和 PhotoShop 模式，开箱即用
- 🔄 完整的颜色变化事件回调支持
- 📐 多种预设样式和自定义选项
- 🖱️ 支持滑块、色块等多种交互方式

## API

### 通用属性

| 参数名称         | 说明               | 类型                                                      | 默认值 |
| ---------------- | ------------------ | --------------------------------------------------------- | ------ |
| color            | 组件的颜色状态值   | `string` \| [Hlsa](#hlsa) \| [Rgba](#rgba) \| `undefined` | ——     |
| onChange         | 颜色值变化时触发   | `(v) => void`                                             | ——     |
| onChangeComplete | 颜色变化完成时触发 | `(v:string) => void`                                      | ——     |

### 默认 Sketch 模式属性

| 参数名称  | 说明                       | 类型                 | 默认值             |
| --------- | -------------------------- | -------------------- | ------------------ |
| className | 容器类名                   | `string`             | `pui-color-picker` |
| initColor | 初始颜色，触发按钮的默认色 | `string`             | ——                 |
| onCancel  | 点击取消时的回调函数       | `(v) => void`        | ——                 |
| onConfirm | 点击确定时的回调函数       | `(v:string) => void` | ——                 |

### Photoshop 模式属性

| 参数名称  | 说明                       | 类型                 | 默认值             |
| --------- | -------------------------- | -------------------- | ------------------ |
| className | 容器类名                   | `string`             | `pui-color-picker` |
| initColor | 初始颜色，触发按钮的默认色 | `string`             | ——                 |
| onCancel  | 点击取消时的回调函数       | `(v) => void`        | ——                 |
| onConfirm | 点击确定时的回调函数       | `(v:string) => void` | ——                 |
| title     | 面板顶部名称               | `string`             | Color Picker       |

### Twitter 模式属性

| 参数名称 | 说明                 | 类型                            | 默认值                                                                                                           |
| -------- | -------------------- | ------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| width    | 单个色块的宽度(px)   | `string`                        | `276px`                                                                                                          |
| colors   | 默认展示的颜色值集合 | `string[]`                      | `['#FF6900', '#FCB900', '#7BDCB5', '#00D084', '#8ED1FC', '#0693E3', '#ABB8C3', '#EB144C', '#F78DA7', '#9900EF']` |
| triangle | 小三角形位置         | `hide \| top-left \| top-right` | ——                                                                                                               |

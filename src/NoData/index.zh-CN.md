## API

| 参数名称  | 说明               | 类型     | 默认值             |
| --------- | ------------------ | -------- | ------------------ |
| className | 样式类名           | `string` | `pui-nodata`       |
| emptyText | 无数据时的文字描述 | `number` | ——                 |
| src       | 无数据时展示的图片 | `string` | 内置的 base64 图片 |

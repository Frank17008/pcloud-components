# pcloud-components

pcloud-components 是一套基于 Antd v4.19.3 + React v17.0.0 开发的业务组件库。

## 开始

安装依赖

```bash
$ npm i
```

启动服务

```bash
$ npm start
```

构建文档

```bash
$ npm run docs:build
```

构建库

```bash
$ npm run build
```

发布文档站点

```bash
$ npm run deploy
```

构建 Docker 镜像

```bash
$ npm run docker:build
```

推送 Docker 镜像

```bash
$ npm run docker:push
```
